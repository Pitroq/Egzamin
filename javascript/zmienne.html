Zmienne

Zmienna to pudełko, które przechowuje jakąś wartość. Aby zadeklarować zmienną należy użyć polecenia var. Zmienna nie może zaczynać się cyfrą.

Składnia:

var liczba = 69;
var string = "tekst";
var bool = true;

Aby wypisać zmienne w konsolimożemy użyć funkcji console.log()

console.log(liczba);
console.log(string, bool);

Aby zobaczyć konsole należy kliknąć prawym przyciskiem myszy i wybrać opcje Zbadaj element lub skrótem Ctrl+Shift+I. Powinny otworzyć się narzędzia developera, wchodzimy w zakładke Console

Przez funkcji console.log() możemy nie tylko wypisywać zmienne na ekran, ale także np. ciągi znaków. Możemy wypisać zmienną i tekst.

console.log("Moja liczba to: ", liczba);
