Wypisywanie wartości

Dobrym sposobem na wypisywanie wartości jest zmienianie wartości danego bloku. Atrybut innerHTML dla znaczników (np. span, div, p) jest tym samym co value dla inputów.

Składnia :

document.getElementById("naszeId").innerHTML = "tekst";

Przykład funkcji zmieniajacej wartość tekstową spana: 

<script>
    function zmienWartosc(){
        var tekst = document.getElementById("tekst").value;
        document.getElementById("span1").innerHTML = tekst;
    }
    </script>

<input type="text" id="tekst">
<input type="button" value="Zmień wartość" onClick="zmienWartosc()">
<br><br>
<span id="span1"></span>

Przykład funkcji dopisywającej tekst do diva: 

<script>
    function zmienWartosc(){
        var tekst = document.getElementById("tekst").value;
        document.getElementById("div1").innerHTML += tekst;
    }
</script>

<input type="text" id="tekst">
<input type="button" value="Zmień wartość" onClick="zmienWartosc()">
<br><br>
<div id="div1"></div>
