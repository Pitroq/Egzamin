Modyfikowanie CSS

Aby zmienić CSS jakiegoś elementu można użyć funkcji document.getElementById(), (id nadajemy w HTML) i następnie odnieść się do atrybutu style. Ta funkcja szuka nam elementu o jakimś identyfikatorze, więc na jednej stronie id nie może sie powtarzać

Składnia :

document.getElementById("naszeId").style;

Przykład funkcji zmieniajacej kolor diva: 

<script>
    function kolorTla(kolor){
        document.getElementById("div1").style.backgroundColor = kolor;
    }
</script>

<input type="button" value="Czerwony" onClick="kolorTla('red')">
<input type="button" value="Niebieski" onClick="kolorTla('blue')">
<br><br>
<div id="div1" style="height: 500px; width: 500px; background-color: blue;"></div>
