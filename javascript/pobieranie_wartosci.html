Pobieranie wartości

Aby pobrać wartość z inputu należy użyć funkcji document.getElementById() (id nadajemy w HTML) i następnie odnieść się do atrybutu value. Ta funkcja szuka nam elementu o jakimś identyfikatorze, więc na jednej stronie id nie może sie powtarzać

Składnia :

document.getElementById("naszeId").value;

Jeżeli chcemy pobrać liczbę to powinniśmy przepuścić naszą wartość przez funkcje parseInt(), która zamienia tekst na liczbę.

var wiek = document.getElementById("wiek").value;
wiek = parseInt(wiek);

// lub

var wiek = parseInt(document.getElementById("wiek").value);

Przykład funkcji z pobieraniem danych: 

<script>
    function przeslijDane(){
        var imie = document.getElementById("imie").value;
        var wiek = document.getElementById("wiek").value;
        wiek = parseInt(wiek);

        console.log("Witaj ",imie,", masz ", wiek, "lat.");
    }
</script>

Podaj swoje imie: <input type="text" id="imie"><br><br>
Podaj swój wiek: <input type="number" id="wiek"><br><br>
<input type="button" value="Prześlij dane" onClick="przeslijDane()">

Pobieranie wartości z radio button

<script>
    function przeslijDane(){
        if(document.getElementById("mezczyzna").checked) console.log("jestes mężczyzną");
        else if(document.getElementById("kobieta").checked) console.log("jestes kobietą");
        else console.log("nie wybrałeś swojej płci!");
    }
</script>

<p>Jaka jest twoja płeć?:

<p>mężczyzna: <input type="radio" id="mezczyzna" name="plec"></p>
<p>kobieta: <input type="radio" id="kobieta" name="plec"></p>
<p><input type="button" value="Prześlij dane" onClick="przeslijDane()"></p>

Pobieranie wartości z checkbox

<script>
    function przeslijDane(){
        if(document.getElementById("pies").checked) console.log("pies");
        if(document.getElementById("kot").checked) console.log("kot");
        if(document.getElementById("chomik").checked) console.log("chomik");
    }
</script>

<p>Jakie masz zwierzę?:

<p>pies: <input type="checkbox" id="pies" name="zwierze"></p>
<p>kot: <input type="checkbox" id="kot" name="zwierze"></p>
<p>chomik: <input type="checkbox" id="chomik" name="zwierze"></p>
<p><input type="button" value="Prześlij dane" onClick="przeslijDane()"></p>
