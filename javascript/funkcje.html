Funkcje

Funkcja zawiera ciąg instrukcji stanowiący blok kodu, który może być wielokrotnie wykorzystany w różnych momentach programu. Funkcja może przyjmować argumenty.

Składnia funkcji bez argumentu:

function wypiszTekst(){ //deklaracja funkcji
    console.log("tekst"); // ciało funkcji
}

wypiszTekst(); // wywołanie funkcji

Składnia funkcji z argumentem:

function wypiszTekst(text){ //deklaracja funkcji i argumentu
    console.log(text); // ciało funkcji
}

wypiszTekst("to jest moj tekst"); // wywołanie funkcji
wypiszTekst("inny tekst");

Funkcja może przyjmować więcej niż 1 argument, jeżeli chcemy zadeklarować ich więcej należey oddzielić je przecinkami (,)

function argumentu(arg1, arg2, arg3){

Funkcje możemy przypisać do przycisku. Jeżeli klikniemy przycisk to funkcja sie uruchomi.

<input type="button" onClick="wypiszTekst('uruchomiono funkcje')" value="Uruchom funkcje">
