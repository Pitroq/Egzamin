Pętle for i while

Pętla for pozwala na powtórzenie bloku kodu określoną liczbę razy.

Składnia pętli for

for(_A_; _B_; _C_){
    // powtarzajacy sie kod
}

W części _A_ nadajemy początkową wartość zmiennej

W części _B_ jest warunek - pętla będzie się wykonywać jeżeli warunek zwróci wartość True

W części _C_ zwiększamy lub zmienjszamy licznik zmiennej

Pętla for może się przydać gdy np. chcemy wypisać liczby

 for($i=0; $i<10; $i++){
    echo $i.", ";
}

Pętla while pozwala na powtórzenie bloku kodu, gdy warunek jest prawdziwy.

Składnia pętli for

while(_A_){
    // powtarzajacy sie kod
}

W części _A_ jest warunek - pętla będzie się wykonywać dopóki warunek zwraca wartość True

$i = 0;
while($i < 10){
    echo $i."-";
    $i += 1;
}
