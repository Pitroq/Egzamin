Instrukcje warunkowe

Instrukcji warunkowej używa się do podjęcia decyzji, jeżeli warunek zwróci wartość True (będzie prawdą) wykona się kod

Składnia prostego warunku:

if($a == $b){
    // akcja
}

Składnia kilku warunków:

if($a == 1){
    // akcja1
}
else if($a == 3){
    // akcja2
}
else if($a == 3){
    // akcja3
}
else{
    // akcja4
}

else if - w przeciwnym wypadku jeżeli. Warunek zostanie sprawdzony jeżeli wcześniejsze warunki przyjeły wartość false.

else - w przeciwnym wypadku. Ta część kodu wykona się jeżeli wcześniejsze warunki przyjeły wartość false.

Operatory porównania:

  • == - równe

  • === - identyczne

  • != - nie równe

  • !=== - nie identyczne

  • < - mniejsze

  • <= - mniejsze lub równe

  • > - większe

  • >= - większe lub równe

Przykład warunków: 

if($a == 1){
    echo "zmienna a jest równa 1";
}
else if($a == 2){
    echo "zmienna a jest równa 2";
}
else{
    echo "zmienna a nie jest równa ani 1, ani 2";
}

Możemy łączyć wiele warunków w jeden.

AND, && - i
OR, || - lub

if ($a > 0 && $a < 10){
        echo "zmienna a jest większa od 0 i mniejsza od 10";
    }
if ($a == 4 OR $a == 65){
        echo "zmienna a równa 4 lub 65";
    }

Jeżeli nasz kod jest jedno liniowy nie musimy pisać klamr { }.

if($a > $b) echo "zmienna a jest większa od zmiennej b";
