CREATE USER

Składnia:

CREATE USER 'nazwa_uzytkownika'@'host' IDENTIFIED BY 'haslo';

Instrukcja CREATE USER służy do dodawania nowego użytkownika do serwera MySQL. Piszemy CREATE USER 'nazwe_użytkownika', @ a następnie 'nazwe_ hosta'. Trzeba jeszcze nadać hasło, którym użytkownik będzie sie logował do serwera, więc IDENTIFIED BY 'nasze haslo';

Najlepiej dodać użytkownika z poziomu serwera (nie wybierając żadnej bazy danych).

Jeżeli wejdziemy w Konta użytkowników zobaczymy listę użytkowników, możemy sprawdzić czy zapytanie zostało dobrze wysłane.
